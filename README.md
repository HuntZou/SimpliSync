# SimpliSync

Sync your text with mutiple platform.

github page release:

- Step 1:
> Modify source code.
- Step 2:
> Run `flutter run -d edge` for test.
- Step 3:
> Run `flutter build web`.
- Step 4:
> Copy all files from `build/web/` to `docs/`, where gitpage used.
- Step 5:
> Commit and push.
